== Weather app (pwa)

Sample PWA app for weather forecast


== Getting Started

1. Clone the application and run bundle:
       bundle install

2. Create db, although its not required for this app but required for a rails application to run:
       rake db:create

3. Run rails server
       rails s

4. Go to http://localhost:3000/pwa in your browser in mobile view and weather app is ready :)
